# Mobile (Expo)

## Стартиране
```bash
cd mobile
npm install
npm run start
```

### Важно (телефон)
- Backend-ът трябва да е пуснат и достъпен през Wi‑Fi.
- В приложението отвори **Настройки** и задай URL като:
  `http://192.168.X.X:4000` (IP адреса на твоя компютър)

## Камера
Приложението използва `expo-camera` за снимка и праща снимката към backend OCR API.
